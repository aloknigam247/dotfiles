Installation directory  
__Windows__ - Delete %APPDATA%\Code and %USERPROFILE%\.vscode.  
__macOS__ - Delete $HOME/Library/Application Support/Code and ~/.vscode.  
__Linux__ - Delete $HOME/.config/Code and ~/.vscode.
